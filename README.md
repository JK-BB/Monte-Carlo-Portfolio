Monte Carlo Portfolio Simulator
A comprehensive financial modeling tool that uses Monte Carlo simulation to analyze potential investment outcomes. The project downloads historical stock market data, calculates key statistical parameters (drift and volatility), and runs thousands of simulations using Geometric Brownian Motion to visualize the range of possible portfolio values over time.
Key Features:

*Historical data analysis using Yahoo Finance API
*Statistical parameter estimation (mean return and volatility)
*10,000+ Monte Carlo simulations using Geometric Brownian Motion
*Comprehensive visualizations including price paths, distribution analysis, and confidence intervals
*Risk metrics calculation (percentiles, probability of profit, worst/best case scenarios)

Technologies: Python, NumPy, Pandas, Matplotlib, yfinance
